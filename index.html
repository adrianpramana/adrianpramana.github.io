Mechanical keyboard size 75% is the size of a compact or compact mechanical keyboard. In this layout, the Insert, PrtScr, ScrLock, Pause and some function keys are missing. However, they are combined into function key shortcuts that can be accessed using the Fn key shortcut. This type of keyboard is easy to carry everywhere because it is small and light. In certain brands, the size of the keyboard casing is also thinner so that it can be put in a bag easily.

A 65% mechanical keyboard is mid-size between 60% and 75%. At this 65% size, the F1-F12 function keys are gone and replaced with the Fn shortcut key to access these function keys. The only function keys are Page Up, Page Down, Delete, and arrow keys. This type of keyboard is suitable for users who want a mechanical keyboard that is small and compact but does not sacrifice some of the function keys of a normal size keyboard, such as the directional arrow keys.

The 60% mechanical keyboard is a very compact mechanical keyboard without the function keys found in normal size keyboards. Function keys such as the F1-F12, Page Up, Page Down keys, etc. are not available. To access these function keys, a combination of the Fn function key and its shortcut access key is required. For example, Fn + 1 to access F1. This keyboard is usually used for concise and simple typing with Vim or Emacs applications. Can also be used to play games because of its compact size.
